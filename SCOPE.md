# We Bake — Scope Summary
> Верхнеуровневый обзор для проверки · 2026-03-12

---

## Что реализовано

### Страницы (8 шт.)
- [ ] **Home** — Hero + секция доставки + How It Works + превью каталога
- [ ] **Catalog** — полная сетка из 10 продуктов
- [ ] **Schedule Builder** — конструктор расписания с выбором дней и количества
- [ ] **Checkout** — 3 шага: адрес → оплата → подтверждение
- [ ] **Subscription** — управление подпиской (пауза / отмена)
- [ ] **Delivery History** — история доставок с возможностью оставить оценку
- [ ] **Billing** — история платежей с раскрываемой детализацией
- [ ] **Waitlist** — страница ожидания для неактивных районов

---

### Глобальные элементы
- [ ] **Шапка** — логотип, навигация, выбор района, авторизация
- [ ] **Баннер** — информирует о текущем районе, можно сменить или скрыть
- [ ] **Выбор района** — модальное окно, 4 района (только Georgetown активен)

---

### Модальные окна (3 шт.)
- [ ] **Auth** — magic link + Apple + Google (демо-режим)
- [ ] **Pause/Cancel** — пауза или отмена подписки
- [ ] **Rating** — пошаговая оценка: курьер + каждый продукт отдельно

---

### Секция "City Request" на главной (последние правки)
- [ ] Левая карточка: Georgetown, DC — статус "Now Active", детали доставки
- [ ] Правая карточка: форма "Not in Georgetown?" с полями:
  - City
  - Street
  - ZIP
  - Email
- [ ] После отправки — экран подтверждения

---

## Продукты (10 позиций)
Classic Baguette · Sourdough Loaf · Uzbek Flatbread · Ciabatta · Whole Wheat · Rye Bread · Focaccia · Croissant · Multigrain Roll · Challah

Цены: $2.50 — $6.00

---

## Ценообразование
```
Итого/неделя = Σ (цена × кол-во × дней)
Доставка     = уникальные дни доставки × $2
Первая неделя = −50%
```

## Адаптивность
- Mobile < 768px
- Tablet 768–1023px
- Desktop ≥ 1024px

Все страницы и компоненты адаптированы под все три размера.
